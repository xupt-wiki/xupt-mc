# 邮立方 / Cube of Posts

邮立方 / Cube of Posts 是西安邮电大学校内的 Minecraft 交流团体，于2022年10月12日建立。

## 官网构建

### 安装

```sh
pnpm i
```

### 升级

```sh
pnpm update --save
```

### 运行

```sh
hexo clean
hexo server --draft
```

### 部署

```sh
git add .
git commit -m "update"
git push
```
